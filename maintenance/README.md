# maintenance

维护类技能目录。用于存放项目管理、代码维护相关的技能。

## 技能列表

| 技能 | 描述 |
|------|------|
| [github-project-maintenance](github-project-maintenance/) | GitHub 项目维护 —— 规范对自己项目和他人项目的修改流程，包括分支管理、冲突解决和提交规范 |

## 分类说明

本目录包含以下维护场景的技能：

- **GitHub 项目维护**：覆盖两类场景
  - 对自己项目：直接在 main 分支修改，按规范提交
  - 对他人项目：创建分支 → 判断类型 → 修改提交 → 检查远程 → 解决冲突 → 推送 PR
